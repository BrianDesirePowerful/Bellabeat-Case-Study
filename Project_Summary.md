# Project Summary

## Overview

This project analyzes Bellabeat smart device fitness data to identify user activity trends and provide marketing recommendations.

## Business Problem

Bellabeat wanted to better understand how customers use smart fitness devices in order to improve customer engagement and support data-driven marketing decisions.

## Dataset

- Fitbit Fitness Tracker Dataset (Kaggle)
- Public dataset containing daily activity, sleep, calories, heart rate, and weight tracking information.

## Tools Used

- SQL (BigQuery)
- Google Sheets
- Looker Studio

## Analysis Process

1. Cleaned and prepared raw datasets.
2. Imported data into BigQuery.
3. Wrote SQL queries to analyze activity patterns.
4. Built dashboards in Looker Studio.
5. Developed business recommendations based on findings.

## Key Findings

- Users were grouped into Low, Moderate, and High activity segments.
- Moderate activity users represented the largest customer group.
- Users with higher activity generally burned more calories.
- Daily activity trends revealed opportunities for personalized engagement.

## Outcome

The project demonstrates practical skills in data cleaning, SQL querying, exploratory analysis, dashboard creation, and business storytelling.
