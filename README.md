# Bellabeat Case Study

## Project Overview

This case study analyzes smart device fitness data for Bellabeat to identify customer activity patterns and provide data-driven marketing recommendations.

## Business Problem

Bellabeat wanted to understand how consumers use smart fitness devices in order to improve marketing strategies and encourage greater engagement with Bellabeat products.

## Key Insights

- Users were segmented into Low, Moderate, and High activity groups.
- Moderate activity users represented the largest customer segment.
- Daily activity patterns revealed opportunities for personalized engagement.
- Data visualization supported marketing recommendations.

## Business Recommendations

- Target moderate users with engagement campaigns.
- Encourage low activity users through wellness challenges.
- Personalize notifications using activity data.
- Develop campaigns focused on healthy habit formation.
